# Task 3 - Image Classification using CNN (CIFAR-10)

## Objective
Build a Convolutional Neural Network (CNN) for image classification.

## Dataset
Name: CIFAR-10  
Source: TensorFlow/Keras built-in dataset  

## Description
A CNN model is trained to classify images into 10 categories such as airplanes, cars, birds, etc.

## Techniques Used
- Deep Learning (CNN)
- Image preprocessing
- Model training and evaluation

## Deliverable
Functional model with performance evaluation on test data.

## How to Run
